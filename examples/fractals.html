Sierpinski Triangle is one of the simplest fractals. Its construction was introduced by the Polish mathematician Wacław Sierpiński in 1915. It is created by drawing an equilateral triangle, connecting the midpoints of its sides to form 4 smaller triangles. Then, the central triangle is removed, and the same process is repeated for the remaining 3 triangles.

Mandelbrot Set is one of the most recognizable objects in contemporary mathematics. The name of this object was introduced to honor its discoverer, the mathematician Benoit Mandelbrot. Mandelbrot popularized fractal geometry, raising awareness that fractals can take shapes similar to natural ones.

The Mandelbrot Set is defined through the iteration of complex numbers according to the equation:

zn+1 = zn2 + c

where c is a complex number from the set, and the value z0 = 0. Numbers for which the iterated sequence defined in this way does not tend to infinity belong to the Mandelbrot Set.

Julia Set and Fatou Set are two complementary sets (each other's complements) defined by a mapping that is a rational function. Informally, the Fatou set of a function contains values with the property that in their vicinity, other values behave similarly under iterated transformations by the specified function. On the other hand, in the Julia set, values are those for which any small perturbation can cause drastic changes in the sequence of iterated values. Therefore, the behavior of the function in the Fatou set is 'regular,' while in the Julia set, it is 'chaotic.'

The Julia Set, like the Mandelbrot Set, is a subset of the complex plane. The name of the set comes from the French mathematician Gaston Julia. The set is composed of those points p for which the sequence described by the recursive equation:

zn+1 = zn2 + c

does not tend to infinity, where c is a complex number serving as the parameter of the set, and z0 = p. As you can see, the equation is very similar to the one we used to calculate the Mandelbrot set. However, this time, c is constant, and we only change the initial parameter of the equation. Earlier, we started with z0 = 0, and now it is equal to p, which is the point we want to check if it belongs to the set.

Fractals in nature

Structures with a fractal-like nature are also commonly encountered in nature. Examples include crystalline dendrites (e.g. snowflakes), water systems of rivers, lightning, or aloe vera flowers.
